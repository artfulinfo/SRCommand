SRCommand

Use the game and category alias in the command format shown above to search for PBs in that game.

The SRDC username is optional, and will default to the username setup by the streamer if one isn't provided. If only a game is provided, a list of pbs in enabled categories will be returned. Note: this will only work for the streamer's runs.

Command

!pb game_alias [category_alias] [srdc_username]
